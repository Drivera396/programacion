DESCRIPCION DEL PROYECTO 
1. Descripción del proyecto 
El proyecto consiste en aplicar los principios de la Programación Orientada a Objetos (POO) para modelar y simular el funcionamiento de un monitor de ECG 
(electrocardiograma), integrando además la lectura, procesamiento y visualización de datos cardíacos. Se implementa una arquitectura basada en tres capas: 
la clase Main, que coordina el flujo general del programa; la clase ArchivoDatos, que gestiona la obtención y carga de datos desde una fuente local o externa; 
y la clase Graficador, que permite representar visualmente las señales cardíacas como taquicardia, bradicardia o normalidad. Esta estructura modular facilita la
interacción entre los componentes, asegurando un manejo eficiente de la información desde su origen hasta la visualización gráfica. Asimismo, 
el sistema incorpora una base de datos externa como repositorio de registros históricos, que sirve tanto para simular lecturas en ausencia de un paciente como 
para validar resultados reales en distintos escenarios clínicos. De esta manera, se logra un modelo flexible, escalable y realista que combina la simulación de 
señales con su análisis visual.
